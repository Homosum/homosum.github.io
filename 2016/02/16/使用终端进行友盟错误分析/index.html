使用终端进行友盟错误分析

  • 打开Xcode打包界面
    Xcode -> Window -> Organizer
  • 找到对应包并找到分析文件
    ipa -> 右键 -> showInFinder

里面有个dSYMs文件夹，里面即使需要用到的dSYM文件

  • 打开终端执行以下语句
    dwarfdump --arch=arm64 --lookup 0x100047880 "$dSYMPath"
  • 其中0x100047880在友盟的错误分析页使用蓝绿色标出的地址 码 “$dSYMPath” 代表dSYM文件路径，删除掉将前面的dSYM文件拖进终端即可。
